## Run Locally

Clone the project

```bash
  git clone https://github.com/cjette98/PokemonMobileApp/
```

Go to the project directory

```bash
  cd PokemonMobileApp
```

Install dependencies

```bash
  yarn install
```

Run the app, Make sure you have the emulator / connected device

```bash
  yarn run android
```
